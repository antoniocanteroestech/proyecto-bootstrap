Modals

The modal text-dark is a dialog box/popup window which can be used for lightboxes, user notifications, UI enhancements, e-commerce components and many other cases. It's easily customized. You can manipulate size, position, and content.

Using the grid

Utilize the Bootstrap grid system within a modal text-dark by nesting .container-fluid within the .modal-body. Then, use the normal grid system classes as you would anywhere else.

Varying modal text-dark content

Have a bunch of buttons that all trigger the same modal text-dark with slightly different contents? Use event.relatedTarget and HTML data-* attributes (possibly via jQuery) to vary the contents of the modal text-dark depending on which button was clicked.
