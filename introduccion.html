Introduction to Bootstrap

On this page we will shoy you a short introduction to bootstrap in which we will explain a little about what it is and how it works. We will also show how to install it and initialize it in your proyect to start working whit this fantastic framework.

What is bootstrap?

Bootstrap is framework created by Twitter and its purpose is to facilitate web design because it allow the creation of an adaptable website, that is, that fits any screen and is always see correctly, in a simple way. This framework is open source so it´s totally free.

Why use Bootstrap?

The vast majority of people are concerned that visiting their website is a pleasant and attractive experience, and that it does not matter from the device or screen that visit it, and Bootstrap makes this task lees complex.

Advantage

  • The learning curve makes its management affordable and fast if you already know how to design.
  • Allows you to use many web elements: from icons to drop-down, combining HTML5, CSS and JavaScript.
  • The page will be adaptable to any scale or resolution regardless of the content.
  • With grid System it is very easy to design by columns and they are very configurable.
  • Its integration with the main JavaScript libraries is very good.
  • It has a very active community improving Bootstrap continuously.
  • It has external implementations (WordPress, Drupal, etc)
