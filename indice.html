Indice de contenidos

A continuación se establecen las distintas seccionesj y una aproximación a sus contenidos. De vosotros y vuestra capacidad de síntesis y decisión depende el resto.

Introducción

¿Qué es Bootstrap? Además, en este apartado se deberán abordar otros aspectos como la instalación y configuración de bootstrap en nuestro proyecto.

Contenedores y sistema de rejilla

El sistema de contenedores y distribución de contenidos en columnas es uno de los puntos fuertes de bootstrap, ya que lo convierte en una herramienta bastante flexible para el diseño de nuestros sitios web. Además, en la versión 4 integra Flexbox.

Diseño Responsivo

Mobile first es la tendencia en el diseño web actual. Ver como se integra el diseño responsive al trabajar con Bootstrap es el objetivo de esta sección. Además se pueden comentar algunos aspectos de los @media de CSS)

Navbar

Gran parte de la navegación de un sitio web se centraliza en el navbar. Bootstrap nos permite que este sea, por supuesto, responsivo. Además, hay otros elementos que ayudan a la navegación com el breadcrumb.

Elementos CSS

Muchas cosas de las que podíamos hacer con CSS se pueden hacer con este framework. En esta sección veremos un repaso a lo selementos CSS principales para el formateo de textos, márgenes, imágenes, etc... desde el punto de vista de un bootstrapiano.

Paneles

Bootstrap permite establecer distintos tipos de paneles para incluir los distintos bloques de contenido de nuestras vistas.

Tarjetas de presentación

Un sitio web es ante todo una herramienta de transmisión de la información de modo visual. Por ello estos tipos de elementos son muy importantes, ya que nos permiten sintetizar información y presentarla con una interfaz gráfica intuitiva y comprehensiva.

Modales

Ayudan a establecer una mayor interacción con el usuario. Bootstrap nos facilita su implementación. Para los ejemplos requerirá algo de javascript.

Alertas

Una aplicación web debe guiar a los usuarios hacia la luz, y evitar que sea usada de un modo incorrecto. Para ellos las alertas cumplen un papel esencial. Para los ejemplos requerirá algo de javascript.

Badges

Otro modo en que las aplicaciones web muestran información al usuario, es a través de badges. Para informar de manera rápida y visual el número de notificaciones o emails que están aguardando ser leídas, por ejemplo.

Tablas

La esencia de una aplicación web es manejar información, y esa información debe ser mostrada por pantalla, en gran parte de las ocasiones por medio de tablas.

Listas

Otro modo de mostrar conjuntos de datos es a través de listas.

Carrousel y Jumbotron

Estos dos elementos nos permiten presentar imágenes de distintos modos, según nuestras necesidades.

Scrollspy

Vamos a dotar a nuestro sitio web de esta funcionalidad, para que la navegación sea aún más vistosa y funcinal.

Formularios

Son la entrada de información del usuario, y por tanto esenciales en cualquier sitio o aplicacion web.

Si hablamos de comunicación audiovisual, a nuestro sitio web le hace falta integrar de un modo ágil este tipo de contenidos. Bootstrap también tiene ayudas para ello.
